# 📊 Sales Data Analysis & Business Intelligence Project

## 📌 Project Overview

This project is a part of my internship at Main Crafts Technology. This project demonstrates an end-to-end data analysis workflow using **SQL, Excel, and Power BI**. The objective is to analyze sales data, generate insights, and support business decision-making.

---

## 🔄 Project Workflow

1. Data Collection – Orders & Customers tables
2. Data Cleaning – Excel preprocessing
3. Data Integration – SQL JOIN
4. Data Analysis – KPI calculations
5. Data Visualization – Power BI Dashboard
6. Insights – Business Report

---

## 🛠️ Tools & Technologies

* **SQL** – Data merging and querying
* **Excel** – Data cleaning
* **Power BI** – Dashboard creation

---

## 🔗 SQL Implementation

Two datasets were used:

* Orders Table
* Customers Table

---

### Operations Performed:

* INNER JOIN for data merging
* Aggregation (SUM, COUNT)
* KPI calculations
* Grouping and filtering
